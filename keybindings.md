# Keybindings

`h` `j` `k` `l` - Navigation ( `←` `↓` `↑` `→` )

`Enter` - Set the selected wallpaper

`g` - Scroll to the top

`G` - Scroll to the bottom

`f` - Change wallpaper folder

`r`  - Re-cache wallpapers

`R` - Set a random wallpaper

`s`  - Include/exclude subfolders

`.` - Include/exclude hidden files and folders

`/` - Search

`?` - Help

`q`- Quit (`Esc`)
